Enter a URL and view all redirects that are occuring on visit

== Description ==

Debugging redirects can be a real pain, especially when working with hundreds or more. This is a simple tool that allows you to input a URL, and view all redirects ( and subsequent redirects ) that occur. This plugin works independently of any server configuration, or redirection plugin.

== Installation ==

1. Upload the `redirect-spaghetti` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Tools > Redirect Spaghetti and enter URL to view redirects
